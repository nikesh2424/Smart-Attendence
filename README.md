# Smart-Attendence
This project contains automatic attendence system using RFID card to scan the user and store in Excel in CSV format

Hardware Requirements:
Aurdino
RFID Module
RFID card or tag
Breadboard
jumper wires
LCD

Circuit Connection:
RFID RC522 to	Arduino Uno/Nano
SDA          	D10
SCK          	D13
MOSI        	D11
MISO        	D12
IRQ	        Not connected
GND          	GND
RST          	D9
3.3V	        3.3V

LCD to	Arduino 
VCC    	5V
GND    	GND
SDA    	A4
SCL    	A5
